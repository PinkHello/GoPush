**GoPush**
> **GoPush**是开源的分布式推送消息服务.
> * 基于**netty**开发
> * 依赖**redis**
> * 依赖**Java8**
> * 依赖**SpringBoot**
> * 依赖**Zookeeper**
> * 不存在单点问题
> * 拓展开可以做成客服、聊天、推送等诸多系统

- Github: https://github.com/pinkhello/gopush
- 码云: https://gitee.com/openWolf/gopush
